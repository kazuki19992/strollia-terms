# すとろりあ プライバシーポリシー

最終更新日：2026年6月10日

クシダ柔工（以下「運営者」）は、運営者が提供する「すとろりあ」（以下「本アプリ」）における利用者情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

# **1. 基本方針**

すとろりあは、利用者自身が移動履歴を記録し、振り返るためのライフログアプリです。

本アプリは、利用者の移動履歴を運営者が収集することを目的として提供するものではありません。

本アプリは、可能な限り利用者自身の管理のもとでデータを取り扱うことを基本方針としています。

ただし、本アプリの機能提供、サービス改善、不具合解析その他運営上必要な場合には、本ポリシーに従って利用者情報を取り扱います。

# **2. 取得する情報**

本アプリは、以下の情報を取得する場合があります。

## **位置情報**

本アプリは、移動履歴の記録および表示のため、利用者の許可に基づき位置情報を取得する場合があります。

取得される情報には以下が含まれる場合があります。

- 緯度
- 経度
- 高度
- 速度
- 移動方向
- 位置情報の取得時刻

## **利用状況情報**

本アプリは、品質向上、不具合調査およびサービス改善のため、以下の情報を取得する場合があります。

- アプリのバージョン
- OSのバージョン
- 端末種別
- クラッシュ情報
- エラーログ
- 利用機能に関する情報

## **お問い合わせ情報**

利用者がお問い合わせを行った場合、氏名、メールアドレスその他利用者が任意に提供した情報を取得する場合があります。

# **3. 情報の利用目的**

取得した情報は、以下の目的で利用します。

- 本アプリの機能提供
- 移動履歴の記録および表示
- 統計情報および実績機能の提供
- 不具合の調査および修正
- サービス品質の向上
- 利用状況の分析
- お問い合わせ対応
- 不正利用への対応
- その他本アプリの運営に必要な業務

# **4. 情報の保存場所**

本アプリで記録される移動履歴、統計情報、実績情報その他の利用者データは、利用者の端末内に保存される場合があります。

保存場所および保存期間は、本アプリの機能、利用者の設定および利用環境によって異なる場合があります。

# **5. 外部送信について**

本アプリは、機能提供、決済処理、品質向上、不具合解析、利用状況分析その他本アプリの運営に必要な範囲で、利用者情報を外部サービスまたはAPIへ送信する場合があります。

送信される情報の内容は、利用する機能またはサービスによって異なります。

移動履歴または位置情報そのものを外部へ送信する機能を提供する場合、運営者は原則として利用者が明示的に有効化した場合にのみ当該機能を利用可能とします。

ただし、法令に基づく場合または本アプリの提供に必要な場合はこの限りではありません。

# **6. 外部サービス**

本アプリは、機能提供、決済処理、品質向上、不具合解析、利用状況分析その他の目的のため、第三者が提供するサービスまたはAPIを利用する場合があります。

これらのサービスまたはAPIは、サービス提供に必要な範囲で利用者情報、端末情報その他の情報を取得または処理する場合があります。

取得される情報の内容および取り扱いについては、各サービス提供事業者の定める利用条件およびプライバシーポリシーに従います。

運営者は、将来的な機能追加、変更または廃止に伴い、利用するサービスまたはAPIを追加、変更または削除する場合があります。

# **7. 第三者提供**

運営者は、次の場合を除き、取得した情報を第三者へ提供しません。

- 利用者の同意がある場合
- 法令に基づく場合
- 人の生命、身体または財産の保護のために必要な場合
- 公的機関から適法な要請を受けた場合
- 事業承継その他の法的手続に伴う場合

# **8. 安全管理**

運営者は、取得した情報について、不正アクセス、漏えい、改ざんまたは滅失の防止に努めます。

ただし、インターネット通信および電子的保存手段に絶対的な安全性は存在しないため、その完全性を保証するものではありません。

# **9. 利用者による設定変更**

利用者は、端末の設定により位置情報へのアクセス許可その他本アプリに付与した権限を変更または取り消すことができます。

これにより、本アプリの一部または全部の機能が利用できなくなる場合があります。

# **10. 未成年者の利用**

未成年者が本アプリを利用する場合は、保護者または法定代理人の同意を得た上で利用してください。

# **11. プライバシーポリシーの変更**

運営者は、必要に応じて本ポリシーを変更できるものとします。

変更後のプライバシーポリシーは、運営者が公開した時点で効力を生じるものとします。

利用者は、本アプリを利用する際に最新のプライバシーポリシーを確認するものとします。

本ポリシーに関するお問い合わせは、運営者が別途定める問い合わせ窓口までご連絡ください。
