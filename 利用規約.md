# すとろりあ 利用規約

最終更新日：2026年6月10日

本利用規約（以下「本規約」）は、クシダ柔工（以下「運営者」）が提供する「すとろりあ」（以下「本アプリ」）の利用条件を定めるものです。

利用者は、本アプリを利用することにより、本規約に同意したものとみなされます。

# **第1条（適用）**

1. 本規約は、本アプリの利用に関する運営者と利用者との間の一切の関係に適用されます。
2. 運営者は、本アプリに関して個別のルール、ガイドラインまたは説明事項を定める場合があります。これらは本規約の一部を構成するものとします。
3. 本規約と個別のルール等の内容が異なる場合は、個別のルール等が優先して適用されます。

# **第2条（未成年者による利用）**

1. 未成年者が本アプリを利用する場合は、事前に法定代理人の同意を得るものとします。
2. 未成年者が法定代理人の同意を得ずに本アプリを利用したことにより生じた損害について、運営者は責任を負いません。
3. 未成年者が本アプリを利用した場合、法定代理人の同意を得ているものとみなします。

# **第3条（本アプリの内容）**

1. 本アプリは、利用者の移動履歴をGPSその他の位置情報を利用して記録・表示するライフログアプリです。
2. 本アプリでは主に以下の機能を提供します。
- 移動経路の記録および表示
- 地図上での履歴表示
- 移動距離、移動時間、速度その他の統計情報の表示
- 実績およびトロフィー機能
- GPX形式によるデータのインポートおよびエクスポート
- その他運営者が提供する機能
1. 運営者は、利用者への事前通知なく本アプリの機能を追加、変更または削除できるものとします。

# **第4条（位置情報の利用）**

1. 本アプリは、利用者の許可に基づき位置情報を取得します。
2. 利用者が位置情報の利用を許可しない場合、本アプリの一部または全部の機能を利用できない場合があります。
3. 本アプリは、移動履歴を記録するため、バックグラウンドで位置情報を取得する場合があります。
4. 利用者は、位置情報の利用許可をいつでも端末設定から変更できます。

# **第5条（記録データ）**

1. 本アプリで記録される移動履歴、統計情報、実績情報その他のデータ（以下「記録データ」）は、原則として利用者の端末内に保存されます。
2. 運営者は、利用者の記録データを保管する義務を負いません。
3. 利用者は、自己の責任において記録データの管理およびバックアップを行うものとします。
4. 運営者は、以下を含む事由により記録データが消失、破損、欠損または変更された場合であっても責任を負いません。
- アプリの不具合
- OSアップデート
- 端末故障
- ストレージ障害
- アプリの削除
- 利用者または第三者の操作
- GPXデータのインポートまたはエクスポート
- その他理由の如何を問わず発生した事象

# **第6条（位置情報および統計情報の精度）**

1. 本アプリは、利用者の端末が提供する位置情報を利用して動作します。
2. GPS、Wi-Fi測位、携帯電話基地局測位その他の位置測位技術には誤差が含まれます。
3. 運営者は、位置情報およびこれを基に算出される統計情報の正確性、完全性、一貫性または継続性を保証しません。
4. 運営者は、以下を含む位置情報由来の事象について責任を負いません。
- 実際とは異なる位置の記録
- 実際とは異なる移動距離の記録
- 実際とは異なる速度の記録
- 実際とは異なる移動経路の記録
- 実際には訪問していない地域の記録
- 記録漏れ
- 記録重複
- その他位置情報の誤差に起因する事象
1. 利用者は、本アプリで表示される位置情報、移動履歴および統計情報が参考情報であることを理解した上で利用するものとします。

# **第7条（実績および統計機能）**

1. 本アプリは、実績、トロフィー、訪問地域数、移動距離その他の統計機能を提供する場合があります。
2. これらの情報は、本アプリ独自の判定基準および集計方法に基づいて算出されます。
3. インポートされたデータの取り扱い、実績判定の対象範囲、統計情報の集計方法その他の仕様は、運営者の判断により変更される場合があります。
4. 運営者は、実績解除の正確性、訪問地域判定の正確性および統計情報の正確性を保証しません。
5. 運営者は、実績の解除漏れ、誤解除、消失、集計漏れ、重複集計その他の不整合について責任を負いません。
6. 実績および統計機能は娯楽目的で提供されるものであり、公的記録、業務記録または証明資料としての利用を想定していません。

# **第8条（バッテリー消費等）**

1. 本アプリは位置情報を継続的に利用するため、通常のアプリと比較してバッテリー消費が増加する場合があります。
2. 利用者は、本アプリの利用により発生する電力消費、通信量、端末発熱その他の影響を理解した上で利用するものとします。
